Course Syllabus

What You Will Learn

Master Lookup functions (VLOOKUP, XLOOKUP), Pivot Tables, Macros, and Data Validation. Learn to create interactive dashboards within Excel.

Learn Relational Database concepts. Master SELECT, WHERE, JOINs, Group By, Window Functions, and Subqueries to extract meaningful data from databases.

Connect to data sources, create Calculated Fields, and build stunning visualizations. Learn to design interactive dashboards and stories for presentations.

Master Power BI Desktop. Learn Power Query for data transformation (ETL), Data Modeling, and DAX functions to create enterprise-level reports.

Learn Python basics, Pandas for data manipulation, NumPy for numerical analysis, and Matplotlib/Seaborn for programmatic visualization.

Understand key concepts like Mean/Median/Mode, Standard Deviation, Probability Distributions, and Hypothesis Testing for data-driven insights.

Learn techniques to handle missing values, duplicates, and inconsistent data. Master the Extract, Transform, Load (ETL) process crucial for analysis.

Work on 3 major end-to-end projects: Sales Performance Analysis, Financial Forecasting Dashboard, and Customer Churn Analysis.

Resume building, LinkedIn profile optimization, and mock interviews specifically designed for Data Analyst roles.
