Course Syllabus

What You Will Learn

Learn the basics of computer networks, TCP/IP, firewalls, VPNs, and security essentials. Build a strong foundation for ethical hacking and system protection.

Master information gathering, OSINT, and reconnaissance techniques. Use AI-powered tools to automate scanning, mapping, and vulnerability discovery.

Discover system vulnerabilities, perform password attacks, session hijacking, and privilege escalation. Learn to exploit and secure systems using AI-assisted tools.

Understand OWASP Top 10 vulnerabilities like SQL injection, XSS, and CSRF. Use AI-driven penetration testing tools for automated web app security checks.

Work with Wi-Fi encryption protocols, rogue access points, and wireless traffic analysis. Explore AI-powered detection and prevention of wireless attacks.

Dive into viruses, worms, Trojans, ransomware, and reverse engineering. Learn encryption, hashing, and AI-powered malware detection techniques.

Gain skills in digital forensics, evidence collection, and cybercrime investigation. Use AI-driven tools for log analysis, threat hunting, and automated incident response.

Secure cloud platforms like AWS, Azure, and GCP. Learn AI-based compliance monitoring, vulnerability scanning, and cloud threat prevention.

Simulate real-world cyberattacks, penetration tests, and red team–blue team exercises. Work on AI-powered adversarial simulations and hands-on projects.
